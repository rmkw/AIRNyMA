Valida primero tu archivo Excel y después ejecuta la importación para registrar fuentes, variables, relaciones y pertinencia en un solo flujo.

Descargar plantilla

Guía rápida para usar la plantilla

Sigue este flujo para evitar errores antes de importar.

1. Descarga la plantilla. Da clic en “Descargar plantilla” y guarda el archivo en tu computadora.
2. Abre el archivo en Excel y habilita la edición. Si aparece un aviso, da clic en “Habilitar edición”.
3. Pega tu información. Copia y pega tus datos en la columna C, desde la fila 2.
⚠️ No pegues sobre la fila 1 porque contiene encabezados.
4. Verifica columnas automáticas. Revisa que id_fuente y id_a se llenen correctamente.
Estas columnas son obligatorias y no deben modificarse manualmente.
5. Guarda el archivo y súbelo al sistema. Guarda los cambios, selecciona el archivo y luego valida antes de importar.
6. Valida primero, importa después. Espera el mensaje de archivo válido. Si hay errores, corrígelos en Excel antes de continuar.
Si no validas el archivo o si existen errores, la información no se importará.
Tip: si algo falla, revisa primero el detalle de validación antes de volver a subir el archivo.
