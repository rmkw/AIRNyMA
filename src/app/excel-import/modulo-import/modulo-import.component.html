Descargar plantilla

Guía rápida para usar la plantilla

  1. Descarga la plantilla.
    Da clic en “Descargar plantilla” y guarda el archivo.
  2. Abre el archivo en Excel.
    Al abrirlo, da clic en “Habilitar edición”.
  3. No toques las columnas A y B.
    (id_fuente y id_a) se llenan solas.
  4. Pega tu información.
    Copia y pega tus datos en la columna C, desde la fila 2.
  5. Guarda y sube el archivo.
    Guarda el Excel y cárgalo en el sistema para continuar.
Si no habilitas la edición o modificas las columnas A y B, el archivo no se podrá importar.
