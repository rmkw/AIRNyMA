Importación con plantilla.xlsx

Valida primero tu archivo Excel y después ejecuta la importación para registrar fuentes, variables, relaciones y pertinencia en un solo flujo.

Descargar plantilla

Guía rápida para usar la plantilla

Sigue este flujo antes de validar e importar el archivo.

  1. 1

    Descarga la plantilla

    Da clic en Descargar plantilla y guarda el archivo en tu computadora.

  2. 2

    Abre el archivo en Excel

    Si aparece un aviso, selecciona Habilitar edición.

  3. 3

    Pega tu información

    Copia tus datos en la columna C, desde la fila 2.

    No pegues información sobre la fila 1 porque contiene encabezados.
  4. 4

    Verifica columnas automáticas

    Revisa que id_fuente y id_a se llenen correctamente.

    Estas columnas son obligatorias y no deben modificarse manualmente.

  5. 5

    Guarda y sube el archivo

    Guarda los cambios, selecciona el archivo desde el sistema y ejecuta la validación.

  6. 6

    Valida antes de importar

    Espera el mensaje de archivo válido. Si hay errores, corrígelos en Excel antes de continuar.

Si no validas el archivo o existen errores, la información no se importará.

Tip: si algo falla, revisa primero el detalle de validación. Si el problema continúa, contacta al desarrollador del sistema.
